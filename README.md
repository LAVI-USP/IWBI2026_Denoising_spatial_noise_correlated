# Model-Based Denoising of Digital Mammography Incorporating Spatial Noise Correlation

This repository contains the code and supplementary material associated with the manuscript submitted to the *18th International Workshop on Breast Imaging (IWBI 2026)*:

**Brandão RF, Borges LR, Caron RF, Barufaldi B, Maidment ADA, Vieira MAC.**
*Model-Based Denoising of Digital Mammography Incorporating Spatial Noise Correlation.*

## Full Paper

The complete manuscript is available here:

📄 **[Download Full Paper](paper/IWBI2026_FullPaper.pdf)**

## Repository Contents

This repository will include:

* Correlated noise simulation framework for digital mammography;
* Model-based denoising pipelines assuming:

  * spatially uncorrelated noise (AWGN);
  * spatially correlated noise;
* Validation scripts for:

  * MNSE;
  * Bias ($\mathcal{B}^2$);
  * Residual Noise ($\mathcal{R}_n$);
  * Power Spectrum (PS);
  * Channelized Hotelling Observer (CHO).
